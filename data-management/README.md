# Data Management

**Project:** Unveiling META Champions in League of Legends  
**Grade:** 30/30

A data pipeline combining the Riot Games API and web scraping to integrate, validate, store, and query champion statistics for patch 13.24.

Python, Jupyter, pandas, web scraping, Riot Games API, MySQL
