# Data Management

**Project:** Unveiling META Champions in League of Legends: A Data-Driven Analysis of the 13.24 Patch
**Grade:** 30/30

A data pipeline combining the Riot Games API and web scraping to integrate, validate, store, and query champion statistics for patch 13.24.

## Stack

Python, pandas, Selenium, Beautiful Soup, MySQL
