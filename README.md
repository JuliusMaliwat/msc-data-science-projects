# MSc Data Science Projects

University of Milano-Bicocca · 2023–2025

| Course | About |
|---|---|
| [Digital Signal and Image Management](./digital-signal-and-image-management) | Environmental sound classification, low-light object detection, and vehicle image retrieval. |
| [Natural Language Processing](./natural-language-processing) | Emotion recognition in multi-turn conversations through task-adaptive representations and dialogue-level context modeling. |
| [Data Science Lab for Smart Cities](./smart-cities-lab) | Spatial analysis of walkable green equity in Milan through pedestrian networks, population data, and neighborhood clustering. |
| [Deep Learning](./deep-learning) | Semantic segmentation of CamVid urban scenes using 32-class and 11-class models. |
| [Text Mining](./text-mining) | Toxic comment classification and topic modeling on the Jigsaw dataset. |
| [Data Management](./data-management) | API and web data integration for League of Legends META analysis. |
| [Machine Learning](./machine-learning) | Smoking behavior classification and biosignal feature analysis in KNIME. |
| [Data Science Lab](./data-science-lab) | Large-scale multilabel text classification with lightweight and transformer models. |
| [Marketing Analytics](./marketing-analytics) | Customer segmentation, churn prediction, market basket analysis, and sentiment modeling. |
| [Data Visualization](./data-visualization) | Tableau analysis of lifestyle habits and sleep efficiency. |
