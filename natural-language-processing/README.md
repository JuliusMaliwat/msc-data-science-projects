# Natural Language Processing

**Project:** Emotion Recognition in Conversation: A Comparative Study of Representation and Context Modeling
**Grade:** 30/30

A comparative study of task-adaptive representations and dialogue-level context modeling for emotion recognition in multi-turn conversations.

## Stack

Python, PyTorch, scikit-learn, Hugging Face Transformers, Sentence Transformers, Captum
