# Data Science Lab for Smart Cities

**Project:** Urban Green Equity in the 15-Minute City: A Data-Driven Framework for Spatial Justice in Milan
**Grade:** 30/30 cum laude

A spatially explicit framework for evaluating walkable access to green spaces relative to population demand across Milan neighborhoods.

## Stack

Python, GeoPandas, OSMnx, Folium, Shapely, scikit-learn

## Contents

- [`notebooks/`](./notebooks) — spatial grid construction, pedestrian networks, green spaces, population, accessibility metrics, and clustering
- [`report/report.pdf`](./report/report.pdf) — final report
